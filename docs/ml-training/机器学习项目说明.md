# 垃圾分类AI模型训练项目

## 项目概述
本项目用于智慧社区平台的垃圾分类功能，包含文字识别和图片识别两个AI模型的完整训练过程。

## 技术栈
- 深度学习框架：TensorFlow/Keras
- 图像处理：OpenCV, PIL
- 自然语言处理：jieba, scikit-learn
- 数据可视化：matplotlib, seaborn
- API服务：FastAPI

## 垃圾分类标准
- 可回收物：废纸、塑料、玻璃、金属、织物等
- 有害垃圾：电池、灯管、药品、化妆品等
- 厨余垃圾：剩菜剩饭、果皮、蛋壳、茶叶渣等
- 其他垃圾：卫生纸、尿不湿、一次性餐具、烟头等

## 训练流程
1. 数据收集和准备
2. 数据预处理和清洗
3. 特征工程
4. 模型构建
5. 模型训练
6. 模型评估

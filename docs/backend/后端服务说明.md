# 智慧社区用户服务 (Python版)

基于Flask框架的用户服务后端，提供用户认证、用户管理、楼栋查询等功能。

## 技术栈

- **框架**: Flask 3.0
- **数据库**: MySQL 8.0
- **认证**: JWT (PyJWT)
- **密码加密**: bcrypt
- **跨域**: Flask-CORS

## 快速开始

### 1. 环境要求

- Python 3.8+
- MySQL 8.0+

### 2. 安装依赖

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

### 3. 配置数据库

确保MySQL已启动，并创建数据库：

```sql
CREATE DATABASE IF NOT EXISTS smart_community 
DEFAULT CHARACTER SET utf8mb4 
DEFAULT COLLATE utf8mb4_unicode_ci;
```

数据库连接配置在 `config/database.py` 中：

```python
DB_CONFIG = {
    'host': '127.0.0.1',
    'user': 'root',
    'password': '123456',  # 修改为您的密码
    'database': 'smart_community',
    'port': 3306,
    'charset': 'utf8mb4',
    ...
}
```

### 4. 启动服务

**方式一：使用启动脚本（推荐）**

```bash
start.bat
```

**方式二：手动启动**

```bash
# 激活虚拟环境
venv\Scripts\activate

# 启动服务
python app.py
```

服务启动后，访问 http://localhost:8081

## API接口文档

### 认证接口

#### 1. 用户注册
```http
POST /api/v1/{platform_type}/auth/register
Content-Type: application/json

{
    "phone": "13800138000",
    "password": "123456",
    "code": "1234"
}
```

#### 2. 用户登录
```http
POST /api/v1/{platform_type}/auth/login
Content-Type: application/json

{
    "phone": "13800138000",
    "password": "123456"
}
```

#### 3. 刷新Token
```http
POST /api/v1/{platform_type}/auth/refresh
Content-Type: application/json

{
    "refreshToken": "your_refresh_token"
}
```

#### 4. 发送验证码
```http
POST /api/v1/{platform_type}/auth/send-code
Content-Type: application/json

{
    "phone": "13800138000"
}
```

### 用户接口

#### 1. 获取当前用户信息
```http
GET /api/v1/{platform_type}/user/info
Authorization: Bearer {token}
```

#### 2. 更新用户信息
```http
PUT /api/v1/{platform_type}/user/update
Authorization: Bearer {token}
Content-Type: application/json

{
    "nickname": "新昵称",
    "email": "new@example.com",
    "building_id": 1,
    "unit": "1单元",
    "room_number": "101",
    "family_structure": "三口之家"
}
```

#### 3. 更新兴趣标签
```http
PUT /api/v1/{platform_type}/user/interest-tags
Authorization: Bearer {token}
Content-Type: application/json

{
    "interestTags": ["团购", "亲子", "运动"]
}
```

#### 4. 修改密码
```http
PUT /api/v1/{platform_type}/user/change-password
Authorization: Bearer {token}
Content-Type: application/json

{
    "oldPassword": "旧密码",
    "newPassword": "新密码"
}
```

#### 5. 更换手机号
```http
POST /api/v1/{platform_type}/user/change-phone
Authorization: Bearer {token}
Content-Type: application/json

{
    "newPhone": "13900139000",
    "code": "1234"
}
```

### 楼栋接口

#### 1. 获取楼栋列表
```http
GET /api/v1/{platform_type}/building/list?community_id=1
Authorization: Bearer {token}
```

#### 2. 获取楼栋详情
```http
GET /api/v1/{platform_type}/building/{building_id}
Authorization: Bearer {token}
```

#### 3. 获取所有楼栋（无需登录）
```http
GET /api/v1/{platform_type}/building/all
```

### 活动接口

#### 1. 获取活动列表
```http
GET /api/v1/{platform_type}/activities
Authorization: Bearer {token}
```

#### 2. 获取活动详情
```http
GET /api/v1/{platform_type}/activities/{id}
Authorization: Bearer {token}
```

#### 3. 报名活动
```http
POST /api/v1/{platform_type}/activities
Authorization: Bearer {token}
Content-Type: application/json

{
    "activity_id": 1
}
```

#### 4. 新增活动（管理员）
```http
POST /api/v1/admin/activities
Authorization: Bearer {admin_token}
Content-Type: application/json

{
    "title": "社区文化节",
    "description": "丰富社区文化生活",
    "start_time": "2026-03-20 10:00:00",
    "end_time": "2026-03-20 16:00:00",
    "location": "社区活动中心",
    "max_participants": 100,
    "image_url": "http://example.com/image.jpg",
    "status": "active"
}
```

#### 5. 更新活动（管理员）
```http
PUT /api/v1/admin/activities/{id}
Authorization: Bearer {admin_token}
Content-Type: application/json

{
    "title": "社区文化节（更新）",
    "description": "丰富社区文化生活，增加互动环节",
    "start_time": "2026-03-20 10:00:00",
    "end_time": "2026-03-20 17:00:00",
    "location": "社区活动中心",
    "max_participants": 120,
    "image_url": "http://example.com/image_updated.jpg",
    "status": "active"
}
```

#### 6. 删除活动（管理员）
```http
DELETE /api/v1/admin/activities/{id}
Authorization: Bearer {admin_token}
```

#### 7. 上传活动图片（管理员）
```http
POST /api/v1/admin/activities/upload-image
Authorization: Bearer {admin_token}
Content-Type: multipart/form-data

file: [图片文件]
```

### 管理员接口

#### 1. 获取用户列表

## 响应格式

所有API返回统一的JSON格式：

```json
{
    "code": 0,      // 0表示成功，其他表示错误
    "msg": "success", // 提示信息
    "data": {}      // 返回数据
}
```

## 平台类型

API路径中的 `{platform_type}` 支持以下值：
- `pc` - PC Web端
- `h5` - 移动端H5
- `admin` - 管理端

## 注意事项

1. **数据库连接**: 确保MySQL服务已启动，且用户名密码正确
2. **端口占用**: 默认使用8081端口，如被占用请修改 `app.py` 中的端口
3. **跨域配置**: 默认允许 localhost:3000 访问，如需修改请编辑 `app.py` 中的CORS配置
4. **JWT密钥**: 生产环境请修改 `utils/jwt_util.py` 中的 `JWT_SECRET`
